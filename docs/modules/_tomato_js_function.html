poll

    • Defined in function/src/poll.ts:40

    轮询方法，支持同步和异步

    新增于v0.0.16

    脚本举例

      import { poll } from '@tomato-js/function'
  const { poller, cancel } = poll({
    fn: getUser,
    validate: v => v.result === true,
    interval: 1000,
    maxAttempts: 20
  });
  //手动取消轮询
  setTimeout(() => cancel(), 7000);
  //满足validate后的fn后续回调
  poller.then(data => {
    console.log('poller done')
    console.log(data);
  });
