chainer

  • chainer(val: unknown): Chain
    • Defined in queue/src/chainer.ts:31

    链式调用

    新增于v0.0.17

    支持链式调用方法：add,subtract,multiply,divide

    脚本举例

      import { chainer } from '@tomato-js/queue'
  const result = chainer(3)
    .add(4)
    .subtract(5)
    .multiply(10)
    .divide(4)
    .add(10)
    .subtract(5)
    .done();//10
